A out-of-the-box neovim configuration for react and web technologies 🌐

<img style={{marginLeft: -25}} src="https://github.com/pauchiner/ChinerNvim/blob/main/.github/images/img2.png?raw=true" height="333" />

## ✨ Features
- Language server protocol built-in
- Prettier formatter by default 
- Git integration
- File explorer
- Telescope for finding
- Syntax highlighting

## 🛠️ Installation

### 🧾 Requirements
- [Nerd Fonts](https://www.nerdfonts.com/font-downloads)
- [Neovim 0.8.2 or higher](https://github.com/neovim/neovim/releases/tag/v0.8.2)

### ⚡ Quick Start
```bash
# Move to .config directory
$ cd ~/.config

# Back up our current config (Recommended)
$ cp -r nvim nvim.backup

# Clone repository
$ git clone https://github.com/pauchiner/chiner.nvim ~/.config/nvim

# Run, when all plugins are installed, reload it 
$ nvim
```
